 ## Project Setup Guide

01/07/2026   Duke 

Follow the steps below to create and run the project locally.

### Create the project using Vite 

If you clone this repository, you can skip the project creation step.

```bash
npm create vite@latest weekly-schedule -- --template react
```

### Clone This Project (Recommended)
```bash
git clone https://github.com/dukehug/weekly_schedule.git
```

###  Navigate to the project directory

```bash
cd weekly-schedule
```

### Install dependencies

```bash
npm install
```

###  Install Tailwind CSS with Vite

```bash
npm install -D tailwindcss @tailwindcss/vite
```

### Start the development server

```bash
npm run dev -- --host 0.0.0.0
```

Once the server is running, open your browser and access the app via the local network IP or `localhost`.
